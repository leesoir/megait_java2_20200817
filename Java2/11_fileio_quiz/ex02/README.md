## 문제) Reverse String  
> 주어진 문자열을 역순으로 출력하는 프로그램  
> StringBuffer의 reverse 메소드 사용 불가  
> char 배열로 변환하여 구한다.  
  
## 입력  
> 첫 번째 줄에 문제의 개수 정수 n이 주어진다. 두 번째 줄 부터 뒤집을 단어가 주어진다.  
  
## 출력  
> 주어진 단어를 역순으로 출력한다.  
  
**예제 입력>**  
3   
hello  
eclipse  
programming  
  
**예제 출력>**  
olleh  
espilce  
gnimmargorp
